1. Network 는 무엇이며 무엇을 목적으로 하는가?
- Net : 망(그물모양)
- Work : 일하다, 작업하다
- 즉 네트워크란 데이터 교환을 위해 통신장치들을 연결한 통신망

- 네트워크의 주요 목적 3가지
- 1) 자원 공유
- 비용 절감과 효율성을 높이는 역할
- 2) 통신
- 네트워크를 통해 다양한 형태의 통신을 가능하게 함
- 3) 데이터 접근
- 중앙 서버에 저장된 데이터에 네트워크를 통해 여러 사용자에게 안전하고 신속하게 제공

2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
- 네트워크의 종류는 크게 LAN과 WAN으로 나눌 수 있다.

3. LAN 의 특징에 대해서 서술하세요
- 장치와 장치간의 거리가 비교적 가까운 근거리 통신망(LAN 구간)
- 속도가 빠르고 설치 및 관리가 간단
- 자원 공유에 최적화

4. WAN의 특징에 대해서 서술하세요
- 장치와 장치간의 거리가 비교적 멀리 있는 광역 통신망(WAN 구간)
- 여러 LAN으로 구성
- 거리와 상관없이 데이터를 전송할 수 있지만 LAN에 비해 속도나 안정성은 떨어짐

5. InterNet 이란
- lnter : 상호간의 접조사
- 즉 전 세계 컴퓨터들을 하나로 연결하는 거대한 컴퓨터 통신망
- 클라이언트와 서버로 구성
- TCP/IP 라는 기본 프로토콜을 통해 제공

6. 프로토콜은 무엇인가?
- 사전에 약속된 통신을 하기 위한 규칙(통신 규약 및 약속)

- 프로토콜의 기본 요소
- 1) 구문(Syntax)
- 전송하고자 하는 데이터의 형식(Format) ,부호화(Coding) ,신호 레벨(Signal Level) 등을 규정
- 2) 의미(Semantics)
- 두 기기 간의 효율적이고 정확한 정보 전송을 위한 협조 사항과 오류 관리를 위한 제어 정보를 규정
- 3) 시간(Timing)
- 두 기기 간의 통신 속도,메시지의 순서 제어 등을 규정

7. 네트워킹 모델의 목적은 무엇인가?
- 네트워크를 잘 만들기 위해서 사전에 미리 보고 만드는 설계도

- 통신망 구현을 위한 지침서
- 하자 발생시 참고하는 지침서
- 이기종간 장치간 통신을 위해 미리 정의해놓은 표준 프로토콜 모델

8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
- TCP/IP 모델 계층
- 1) 네트워크 인터페이스 계층
- 인터넷을 구성하는 각기 다른 물리적 네트워크의 연결과 관련된 기능 수행
- 2) 인터넷 계층
- 데이터를 전송하기 위한 목적지까지 전송 경로 설정(패킷 분할)
- 3) 전송 계층
- 송신 측과 수신 측 사이에서 데이터 전송
- 4) 애플리케이션 계층
- 송신 측과 수신 측의 응용 프로그램 통신 지원 및 데이터 교환 관련된 서비스 제공

- OSI 계층
- 1) 물리계층 
- 장치간 물리적인 연결
- 2)데이터 링크 계층(본격적인 데이터는 여기서부터!)
- 장치간 데이터 운송방식 결졍
- 3)네트워크 계층
- 경로 탐색
- 4)전송 계층
- 송수신간의 데이터의 신뢰성을 확보
- 5)세션 계층
- 데이터가 통신하기 위한 논리적인 연결
- 6)프레젠테이션 계층
- 사용자의 명령어를 완성 및 결과 표현.포장/압축/암호화
- 7)애플리케이션 계층
- 네트워크 소프트웨어 UI부분, 사용자의 입출력(I/O) 부분
